# gene_clustering
**Кластеризация данных экспрессии генов для классификации заболеваний**
 - Описание проекта
Данный проект направлен на применение методов машинного обучения для анализа данных экспрессии генов с целью выявления закономерностей, связанных с различными заболеваниями, в частности, онкологическими. Основной задачей является использование алгоритмов кластеризации для группировки образцов на основе их профилей экспрессии генов, что позволяет классифицировать заболевания или подтипы заболеваний.

 - Задача проекта
Применить методы кластеризации (иерархическая, k-средних, DBSCAN) к данным экспрессии генов для выявления групп образцов с разными заболеваниями.

В рамках проекта были использованы следующие алгоритмы:
 - K-средних (K-Means): Итеративный алгоритм, разделяющий данные на заранее заданное количество кластеров (K).
 - Иерархическая кластеризация (Hierarchical Clustering): Метод, создающий иерархию кластеров, представленную в виде дендрограммы.
 - DBSCAN (Density-Based Spatial Clustering of Applications with Noise): Алгоритм, основанный на плотности, способный находить кластеры произвольной формы и выделять шум.

Работа над проектом включала следующие ключевые шаги:
 - Нормализация данных: Обработка исходных данных для обеспечения сопоставимости уровней экспрессии между различными образцами.
 - Снижение размерности: Использование методов для уменьшения количества признаков (генов) при сохранении важной информации.
 - Кластеризация: Применение выбранных алгоритмов кластеризации к подготовленным данным.
 - Визуализация: Графическое представление результатов для интерпретации полученных кластеров.
 - Биологический анализ: Интерпретация результатов кластеризации в биологическом контексте для подтверждения их значимости и связи с типами заболеваний.
